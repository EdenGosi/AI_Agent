# ü§ñ AI Agent with Gemini (LangChain + LangGraph)

This project is a **Python-based AI agent** built using **LangChain**, **LangGraph**, and **Google Gemini**.  
It demonstrates how to create an interactive AI assistant that can reason, respond and be extended with tools
---

## Features

- Uses **Google Gemini (free tier)
- Built with **LangChain** and **LangGraph**
- Interactive command-line chat interface
- Secure API key handling using `.env`
- Easy to extend with tools (calculator, memory, search, etc.)
- Dependency management using **uv**

---

## Tech Stack

- Python 3.13
- LangChain
- LangGraph
- Google Gemini API
- python-dotenv
- uv(package & environment manager)
